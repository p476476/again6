# Again System -- Unity AVG 對話系統

## 目標

- 成為比 Fungus 更好用對話系統
- 精簡化 Fungus，省略一堆用不到的指令和工具
- 對企劃人員友善，支援 Google Sheet 製作對話表，遊戲中可以直接讀取播放
- 對程式人員友善，一個 Prefab 隨放及用，一個一目了然的設定介面
